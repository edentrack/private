# How to Add Animal Images

## Images Needed

Please add these images to the `public` folder:

1. **rabbit.png** - Rabbit image (you provided)
2. **catfish.png** - Catfish image (you provided)  
3. **tilapia.png** - Tilapia image (you provided)

## Where to Place Them

Place all three images in: `/public/` folder

The app will automatically use them:
- `rabbit.png` for rabbits
- `catfish.png` for catfish
- `tilapia.png` for tilapia

## Current Images

The app already has:
- `/public/image.png` - Layer hen
- `/public/broiler.png` - Broiler chicken

After adding the new images, the app will automatically display them in:
- Create Flock Modal (type selection)
- Flock Management page (flock cards)
